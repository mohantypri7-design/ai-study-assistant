# AI Study Assistant

## Overview

AI Study Assistant is an AI-powered learning platform that helps students transform PDFs, notes, and study materials into useful learning resources.

The application uses the Google Gemini API to automatically generate:

* Study Summaries
* Flashcards
* Multiple Choice Questions (MCQs)
* Important Exam Questions
* AI Tutor Responses

The goal of this project is to reduce study preparation time and improve learning efficiency.

---

## Problem Statement

Students often spend a significant amount of time:

* Reading large PDFs
* Creating revision notes
* Making flashcards
* Preparing practice questions

This process is repetitive and time-consuming.

AI Study Assistant automates these tasks using Artificial Intelligence.

---

## Features

### PDF & Notes Upload

Upload PDF documents or paste notes directly into the application.

### Summary Generator

Generate structured revision notes and key concepts automatically.

### Flashcards

Create active-recall flashcards from study materials.

### MCQ Generator

Generate multiple-choice questions with answers and explanations.

### Important Questions

Generate exam-style questions with model answers.

### AI Tutor Chat

Ask questions and receive explanations related to the uploaded study content.

### Session Storage

Save study sessions locally for future access.

### Frontend

* React
* TypeScript
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### AI

* Google Gemini API

### Version Control

* Git
* GitHub

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
